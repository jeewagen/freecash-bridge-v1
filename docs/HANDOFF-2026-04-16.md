# Freecash Funnel Handoff

## Current Status

The local funnel is implemented as a static HTML/CSS/JS app with 3 stitched screens and working step logic.

- Page 1 has been heavily refined and is the current approved baseline before moving on to Page 2.
- Page 2 and Page 3 are still functionally working, but have not yet gone through the same visual revision cycle.
- The project is not a git repository, so there is no commit history to rely on for handoff.

## How To Run

From the project root:

```bash

```


## Files That Matter Most

- `src/screenTemplates.js`
  - Contains the stitched HTML templates for Page 1, Page 2, and Page 3
  - Most recent visual work has been concentrated in the Page 1 section at the top of this file
- `src/funnel.js`
  - Controls page switching, timer behavior, and the live footer ticker
- `styles/app.css`
  - Shared visual treatment including glow/pulse effects
- `tests/funnel.test.js`
  - Regression coverage for flow logic and Page 1 design-specific structure

## Page 1 Decisions Locked In

- Welcome bonus pill text is `11px` and remains on one line
- A little more vertical space exists between the welcome bonus row and the rating/heading cluster
- Rating badge sits above the headline
- Supporting copy is:
  - `Earn real money for things you already do on your phone. Cash out instantly via PayPal, Venmo, or CashApp.`
- `RECENT EARNINGS` is aligned with a right-side `Verified Payouts` badge
- Verified badge is bottom-aligned to the section label and uses the same green border token as the testimonial avatar
- Testimonial cards use:
  - smaller circular avatars
  - green payout amounts
  - quote copy spanning the full card width below the header row
- The `21+` button keeps its original base styling with only a soft glow/pulse layered on top
- Footer ticker:
  - sits at the bottom of Page 1
  - uses the project green
  - updates more slowly
  - pulses subtly when the number changes
